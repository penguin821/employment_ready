<struct 와 클래스>는 종이 한장 차이
가장 큰 차이점은 struct는 기본 접근자가 public이고
클래스는 private이다.
그래서 걍 선언하고 접근 하려고 하면 struct는 가능한데 클래스는 불가능

<static 변수, 함수 (정적)>
클래스 클래스명()
{
특정 마린 객체에 종속적
 int hp

특정 마린 객체랑 상관없이
모든 마린이라는 클래스 자체와 연관됨
 static int attack // 설계도상으로만 존재
}
그래서 공격력 방어력 업그레이드 했을때
기존에 이미 생산된 마린이랑 배럭에서 생산되고 있는 마린이 둘다 업그레이드 적용

static 함수를 선언하는 순간
함수 내부에는 static 변수 함수만 사용 가능해짐

클래스명::스태틱변수, 클래스명::스태틱함수 로 선언함
static 변수는 초기화 하면 .data영역
static 변수는 초기화 안하면 .bss영역 (data bss 둘다 전역변수 데이터들 취급)
그래서 스태틱 자료형으로 선언한 변수는 아무리 함수 안에서 선언을 했다고 해도
stack 영역에 데이터가 안올라가고 data 영역에 올라감
따라서 생명 주기도 함수안에 선언해도 프로그램 시작 종료 주기를 따라감
그런데 함수안에 선언했기에 접근이나 호출은 함수 내부에서만 가능함
